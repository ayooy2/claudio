Claudio v2.0 — 架构总览

  54 个源文件，TypeScript 全栈，模块化清晰。

  项目结构

  E:\Claudio\
  ├── server/                          # 后端 (Express + TypeScript + Socket.IO)
  │   ├── src/
  │   │   ├── modules/                 # 8个独立模块，各司其职
  │   │   │   ├── brain/               #   Claude Code 子进程适配器
  │   │   │   ├── context/             #   提示词组装修 (6部分拼接)
  │   │   │   ├── music/               #   网易云音乐 (24首Mock歌曲库)
  │   │   │   ├── tts/                 #   TTS合成 + MD5缓存
  │   │   │   ├── weather/             #   天气服务
  │   │   │   ├── calendar/            #   飞书日历
  │   │   │   ├── player/              #   播放状态管理
  │   │   │   └── scheduler/           #   定时调度 (6个cron任务)
  │   │   ├── routes/                  # 5个路由文件，一对一对应API
  │   │   ├── store/                   # IStore接口 → JsonStore实现 (换数据库只改这里)
  │   │   └── common/                  # logger / errors / middleware
  │   └── prompts/dj-persona.md
  │
  ├── client/                          # 前端 (React 19 + Vite + TailwindCSS 4)
  │   ├── src/
  │   │   ├── components/              # 9个组件: Player/Controls/ProgressBar/...
  │   │   ├── hooks/                   # 3个hooks: usePlayer/useSocket/useApi
  │   │   └── services/api.ts          # API调用封装
  │   └── public/                      # manifest.json + sw.js (PWA)
  │
  ├── docker/                          # Dockerfile (3阶段构建) + docker-compose
  ├── user/                            # 用户品味数据
  ├── package.json                     # npm workspaces 根
  └── .env.example

  为什么这样设计

  ┌────────────┬─────────────────────────────┬──────────────────────────────┐
  │    需求    │            方案             │        换技术时改哪里        │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 换数据库   │ IStore 接口抽象             │ 新增 PgStore.ts 实现接口即可 │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 换前端框架 │ 组件独立于框架              │ 重写 components/ 目录        │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 换TTS服务  │ TtsService 独立模块         │ 改 modules/tts/ 一个文件     │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 拆微服务   │ 模块零依赖，路由协调        │ 把 module 拆成独立 Node 进程 │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 多人访问   │ 当前 JSON 文件 → PostgreSQL │ store/ 新增实现，其余不变    │
  └────────────┴─────────────────────────────┴──────────────────────────────┘

  启动方式

  # 开发模式（前后端热更新）
  cd E:\Claudio
  npm install
  npm run dev          # 同时启动 server:8080 + client:5173

  # 生产部署
  docker-compose -f docker/docker-compose.yml up -d
