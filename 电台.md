  │   └── public/                      # manifest.json + sw.js (PWA)

  为什么这样设计

  ┌────────────┬─────────────────────────────┬──────────────────────────────┐
  │    需求    │            方案             │        换技术时改哪里        │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 换数据库   │ IStore 接口抽象             │ 新增 PgStore.ts 实现接口即可 │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 换前端框架 │ 组件独立于框架              │ 重写 components/ 目录        │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 换TTS服务  │ TtsService 独立模块         │ 改 modules/tts/ 一个文件     │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 拆微服务   │ 模块零依赖，路由协调        │ 把 module 拆成独立 Node 进程 │
  ├────────────┼─────────────────────────────┼──────────────────────────────┤
  │ 多人访问   │ 当前 JSON 文件 → PostgreSQL │ store/ 新增实现，其余不变    │
  └────────────┴─────────────────────────────┴──────────────────────────────┘

5/30：

把之前的规划改成 Claudio 沉浸式像素风音乐播放器
基于原Claudio FM电台升级，纯音乐播放器，仅调用外部API实现播放，极致简洁高级，主打沉浸式场景体验，保留标志性白色方格像素美学。

## 必须保留的灵魂元素
- 顶部左上角：白色点阵像素字体"Claudio"logo
- 界面视觉中心：超大号白色方格像素时:分时钟（与原图完全一致的方块风格）
- 时钟正下方：英文星期 + 日期（纤细半透明白色字体）
- 整体色调：深黑/深色基底 + 白色主UI + 亮绿色点缀
- 半透明悬浮式交互元素，不破坏背景沉浸感

## 核心亮点：沉浸式场景系统（重中之重）
6个可一键切换/自动随系统时间切换的场景，每个场景包含专属动态背景+独立环境音轨：
1. 雨天：深灰蓝渐变，细密白色像素雨滴下落，远处模糊城市灯光 | 轻柔雨声+偶尔闷雷
2. 下午：暖黄色渐变，金色像素阳光斜射，漂浮尘埃粒子 | 轻微咖啡馆白噪音
3. 傍晚：橙紫色渐变，缓慢移动的像素云朵，落日余晖 | 微风声+远处鸟鸣
4. 深夜：纯黑背景，闪烁白色像素星空，偶尔流星划过 | 夜晚虫鸣
5. 森林：深绿色渐变，飘落的像素树叶，林间光斑 | 森林风声+树叶沙沙
6. 极简：纯黑/纯白背景，无动态元素 | 无环境音

- 环境音与音乐独立控制，可单独开关
- 场景切换时带丝滑淡入淡出过渡

## 播放器核心功能（仅API调用）
- 基础播放控制：上一曲/播放暂停/下一曲（三个大圆形半透明按钮）
- 进度条：白色半透明，两端显示当前时间/总时长，支持拖动
- 音量调节：绿色滑块半透明，可自动隐藏
- 播放队列：左侧滑出抽屉式列表，支持点击切歌
- 歌曲收藏：心形图标按钮
- 自动播放下一首

##  极简界面布局（竖版优先，响应式）
[顶部] 左：Claudio像素logo | 右：场景切换按钮 + 全屏按钮
[中心] 超大像素时钟 → 星期+日期
[中部悬浮] 播放控制按钮组 → 进度条 → 音量滑块
[底部] 居中：歌曲名 + 歌手名 | 左：队列按钮 | 右：收藏按钮

- 完全移除：所有电台相关元素（DJ聊天区、输入框、发送按钮、ON AIR标识、播放队列列表直接显示）
- 所有按钮hover时变为不透明，有轻微缩放反馈

## 技术要求
- 所有音乐播放功能通过调用外部音乐API实现
- 动态背景使用纯CSS动画，保证60fps流畅度
- 支持响应式设计，完美适配手机和电脑屏幕
- 支持键盘快捷键（空格播放/暂停，左右箭头切歌）
- 代码结构清晰，API调用接口预留好位置

## 额外加分项
- 支持半透明悬浮歌词（可选显示/隐藏，像素字体）
- 系统时间自动切换场景（12-18点下午，18-20点傍晚，20-6点深夜）
- 歌曲切换时背景有轻微颜色过渡
- 支持PWA安装到桌面
