# 我的音乐品味

## 喜欢的风格
- 华语流行 (周杰伦、林俊杰、陈奕迅、五月天)
- 独立民谣 (赵雷、宋冬野、马頔)
- R&B / Neo-Soul (方大同、袁娅维)
- 轻音乐 / 后摇 (工作背景音)
- City Pop / J-Pop (晚间放松)

## 喜欢的歌手
- 周杰伦 — 初中开始听，每张专辑都熟
- 林俊杰 — 唱功无敌，歌声有力量
- 陈奕迅 — 歌词走心，适合深夜
- 赵雷 — 民谣里的生活气息
- 方大同 — 律动感强，适合周末

## 讨厌的
- 过于嘈杂的重金属
- 抖音神曲 (旋律腻、制作粗糙)
- 喊麦 / 社会摇

- **工作/专注**: 后摇、轻音乐、无歌词电子
- **运动**: 节奏感强的流行、Hip-Hop
- **睡前**: 舒缓民谣、爵士钢琴
- **周末早晨**: Bossa Nova、轻快民谣
- **通勤**: 最近爱听的、随机发现新歌
